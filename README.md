# E-Commerce Website

A full-stack E-Commerce Website developed using Java Full Stack technologies.

## Features
- User Registration & Login
- Product Management
- Shopping Cart
- Order Management
- Admin Dashboard
- Responsive UI

## Technologies Used
- Java
- Spring Boot
- Spring MVC
- Hibernate
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap

## Prerequisites
- JDK 17+
- MySQL
- Maven

## Installation
1. Clone the repository:
   git clone https://github.com/MohammedTarief/e-commerce-web.git

2. Open the project in IntelliJ IDEA.

3. Configure the database.

4. Run the Spring Boot application.
